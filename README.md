# Event Hunt

Event Hunt is a place where you can find the hottest local events

## Installation

2. Install dependencies in client

```
cd event-hunt-frontend
```

```
npm install
```

## Running the application

2. Run the React client

```
cd event-hunt-frontend
```

```
npm run dev
```

## Built With

- [React](https://reactjs.org/)
